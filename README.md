# Alcove Cafe Website

A clean, tab-based website for **Alcove Cafe**.

## Website structure

- Top navigation tabs: **Menu**, **Book Now**, **About Us**, **Contact Us**
- Each tab presents dedicated information specific to that section
- No chess or LAN gameplay features are included

## Run locally

```bash
npm start
```

Then open `http://localhost:3000/`.

## Stack

- Node.js built-in HTTP server
- Vanilla HTML/CSS/JS
